Hola, este es el ejercicio de python, obtener el precio de la estadia en el hotel Asuncion de Madrid.
Para correr el programa se lo debe hacer de esta forma:
python main.py dd1/mm1/aa1 dd2/mm2/aa2 rooms adults child
las fechas usan el formato dd/mm/aa dia, mes, año en números
dd1/mm1/aa1: Fecha de ingreso al hotel
dd2/mm2/aa2: fecha de salida
rooms: Número de abitaciones, por defecto son 1
adults: Número de adultos, por defecto son 2
child: Número de niños, por defecto son 0
